# nooz.rip
A nooz aggregator and reader.

## Usage
- `make install` or `poetry install`
- `make run` or `poetry run python app/run.py`

## `.env` Example
```
DB=nooz.db
HOST=0.0.0.0
PORT=9100
DEBUG=true
JSON_LOGS=false
```
**NOTE:** `DB` is ignored at the moment.
